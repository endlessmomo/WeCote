## 📍 Commit Message Convention
- ADD : 문제 풀이 파일이나 부수적인 코드 추가
    - 파일을 업로드 할 때는 다음과 같은 형식을 따라 주세요
    - ADD : [문제 유형] 백준 ????번 풀이
- FIX : 코드 오류 수정
- MOD : 코드 및 내부 파일 내용 수정
- DEL : 쓸모없는 코드나 파일 삭제
- CORRECT : 문법 오류 해결, 타입 변경, 이름 변경 등의 작은 수정
- RENAME : 파일 이름 변경
